# data/

This folder holds raw source files. **Nothing here is committed to Git.**

## Download instructions

1. Go to https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store
2. Download these two files:
   - `2019-Oct.csv` (~2.5 GB)
   - `2019-Nov.csv` (~3.5 GB)
3. Place them in this folder.

## What happens next

Running `drift/ingestion/upload_to_s3.py` will:
- Convert each CSV to Parquet (columnar format, ~60% smaller)
- Upload to `s3://drift-data-547519648043/raw/`

Parquet on S3 is the source of truth. Local CSVs and Parquet files
are scratch copies and are gitignored.

## Folder contents (gitignored)

```
data/
├── 2019-Oct.csv         ← raw download
├── 2019-Nov.csv         ← raw download
├── 2019-oct.parquet     ← generated by upload_to_s3.py
└── 2019-nov.parquet     ← generated by upload_to_s3.py
```
